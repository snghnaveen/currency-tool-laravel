## Currency Rate Calculator

Sample Laravel web app to use wsdlgenerator and soap requests


### Requirements

- php5+
- php-soap


### Installation
  - Clone the repo
```sh
$ git clone git@github.com:snghnaveen/currency-tool-laravel.git

```
- Change directory
```sh
$ cd currency-convert
```

- Composer install
```sh
$ composer install
```
- Run PHP Server
```sh
$ php artisan serve
```
